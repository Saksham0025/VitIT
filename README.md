VitIT is a social media platform designed specifically for students of Vellore Institute of Technology (VIT).
The platform helps students connect with each other, ask doubts, share queries, and discuss topics related to academics and college life.

It aims to create a collaborative student community where VIT students can support each other with studies, projects, and campus-related discussions.

🚀 Features

💬 Ask doubts related to subjects, assignments, and exams
🧑‍🎓 Connect with other VIT students
📚 Discuss study materials and academic topics
🏫 Share experiences and queries about college life
📝 Post and interact with questions and discussions

🎯 Purpose of the Project

Students often face difficulties such as:
1. Understanding course concepts
2. Finding the right study resources
3. Asking doubts outside the classroom
4. Getting advice about college life

VitIT solves this by providing a dedicated platform where VIT students can help each other.
The goal is to build a student-driven knowledge sharing community inside VIT.

🛠️ Tech Stack

The project is built using modern web development technologies:
Frontend: ReactJS, CSS, JavaScript
Backend: NodeJS and Postgresql
Version Control: Git & GitHub
