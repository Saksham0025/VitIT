VitIT is a digital platform designed to improve student interaction and academic collaboration at Vellore Institute of Technology (VIT).
The platform allows VIT students to connect, ask doubts, share knowledge, and discuss academic or campus-related queries in one centralized space.
VitIT focuses on creating a smart,web app-based campus service that strengthens communication between students and helps them navigate both academic challenges and college life more effectively.
This project aligns with the Tech & Digital Integration Track of the hackathon by providing a digital solution for campus services and student collaboration.

Hackathon Track Alignment:
Track 2: Tech & Digital Integration
VitIT contributes to digital campus transformation by offering:
1. A centralized platform for student interaction
2. A doubt-solving ecosystem for academic queries
3. A collaborative community for sharing resources and experiences
4. A digital support system for navigating college life

By digitizing student discussions and doubt resolution, VitIT aims to become a smart campus communication tool.

✨ Key Features
A. Student Community Platform
  ->Students can interact with peers across different branches and years.
B. Ask & Solve Doubts
  ->Students can post questions regarding:
      1. Subjects
      2. Assignments
      3. Exams
      4. Coding problems
      5. Projects
C. Academic Support
  ->Students can discuss:
      1. Study strategies
      2. Course materials
      3. Lab work
      4. Programming concepts
D. College Life Discussions
  ->Students can ask questions about:
      1. Hostel life
      2. Campus facilities
      3. Clubs & events
      4. Internships and opportunities
E.  Digital Campus Engagement
  ->VitIT acts as a mini digital ecosystem for VIT students to communicate efficiently.

🛠️ Tech Stack
The project is built using modern web development technologies:
->Frontend: ReactJS, CSS, JavaScript
->Backend: NodeJS and Postgresql
->Version Control: Git & GitHub
